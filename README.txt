Integrates Divi with the iko.travel Affiliate WordPress plugin.

== Description ==

This plugin makes adding iko.travel Affiliate web components a breeze with Divi. Simply install and activate it next to the official iko.travel WordPress plugin and get started.

== Frequently Asked Questions ==

= Does it work with the frontend editor of Divi =

Yes. It was built with ReactJS and integrates with all features of Divi.

= Why can I not see the content of the webcomponents when I add the elements with Divi? =

Simply click on save and close the Divi builder by clicking on "Exit Visual Builder".
